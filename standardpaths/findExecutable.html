    Function findExecutable

    Finds executable by fileName in the paths specified by paths.

    Prototype

    string findExecutable(Range)(
  string fileName,
  Range paths
) nothrow @trusted
if (isInputRange!Range && is(ElementType!Range : string));

    Returns

    Absolute path to the existing executable file or an empty string if not found.

    Parameters

    NameDescription
    fileName Name of executable to search. If it's absolute path, this function only checks if the file is executable.
    paths Range of directories where executable should be searched.

    Note

    On Windows when fileName extension is omitted, executable extensions will be automatically appended during search.
