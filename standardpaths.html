    Module standardpaths

    Functions for retrieving standard paths in cross-platform manner.

    Functions

    Name Description
    binPaths System paths where executable files can be found.
    findExecutable Finds executable by fileName in the paths specified by paths.
    findExecutable ditto, but searches in system paths, determined by PATH environment variable.
    homeDir Current user home directory.
    roamingPath Path to Roaming data directory.
    runtimeDir Path to runtime user directory on freedesktop platforms.
    savedGames Location where games may store their saves. This is common path for games. One should use subfolder for their game saves.
    standardPaths Getting paths for various locations.
    writablePath Getting writable paths for various locations.

    Enums

    Name Description

    StandardPath Locations that can be passed to writablePath and standardPaths functions.
